Este repositório centraliza os ativos visuais, a paleta de cores e os princípios de design que compõem a marca **JeanSol**. Este "Design System" garante que todos os dashboards, modelos de machine learning e pipelines de dados sigam um padrão rigoroso de clareza, modernidade e eficiência.

> 💡 **Curioso sobre a construção desta marca?** Confira o nosso [Processo de Design (PROCESS.md)](./PROCESS.md) para ver a evolução desde o rascunho até o logo final.

## 🎨 Paleta de Cores

Nossa paleta foi escolhida para equilibrar a confiança técnica com a energia da resolução de problemas.

| Cor | Nome | Hex | Aplicação Principal |
| :--- | :--- | :--- | :--- |
| `🟦` | **Azure Logic** | `#4d9be6` | Base da identidade, confiança técnica e navegação. |
| `🟨` | **Solar Gold** | `#f9c22b` | Call-to-action (CTAs), destaques de dados e energia da solução. |
| `⬜` | **Pure White** | `#ffffff` | Clareza, espaços negativos e fundo analítico. |

## 📐 Princípios do Engenheiro

1. **Clareza em Primeiro Lugar:** O design deve facilitar a interpretação de métricas complexas, nunca dificultá-la.
2. **Ciência de Dados Aplicada:** Estética e funcionalidade devem servir ao objetivo final: resolver um problema real.
3. **Consistência de Ecossistema:** Um dashboard em Streamlit deve ter o mesmo "feeling" visual que um relatório em Power BI ou uma aplicação de Visão Computacional.
